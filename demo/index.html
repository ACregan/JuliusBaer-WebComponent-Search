Search Web Component

By Anthony Cregan

This is the demonstration page for the <juliusbaer-webcomponent-search> web component. Scroll down to see the component. It has been initialised and provided with mock 'Transaction' data and attributes.

I have added some redacted filler text above and below the search web component, specifically to enable the demonstration of the feature that shows the results panel above the search input when there is not sufficient room on the page below to display them.

To see this in action, simply scroll down and if the search input is nearer the top of the viewport than the bottom then the search results will appear below the search input (and vice versa).

Select the input and type "el" to begin. You will note that all of the results displayed contain "el" and those partially matching values are highlighted amongst the results in yellow.
